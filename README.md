# Deal Studio — דוחות כספיים ← אקסל + מצגת

אפליקציה מקומית: מעלים דוחות כספיים (כמה שנים), Claude קורא אותם ומחלץ את המספרים,
ומקבלים להורדה את **האקסל המלא** (7 גיליונות, נוסחאות חיות) ואת **המצגת המאוכלסת** (12 שקפים) —
אותם תוצרים שבנינו, על נתונים אמיתיים, שוב ושוב.

---

## דרישות מוקדמות
- **Python 3.10+** מותקן (בדיקה: `python3 --version`).
- **מפתח Anthropic API** — מ-https://console.anthropic.com (לשונית API Keys). התשלום לפי שימוש; כל חילוץ דוח עולה אגורות בודדות.

## הפעלה (פקודה אחת)

**macOS / Linux**
```bash
export ANTHROPIC_API_KEY=sk-ant-xxxxx
./run.sh
```

**Windows**
```bat
set ANTHROPIC_API_KEY=sk-ant-xxxxx
run.bat
```

הסקריפט מתקין את התלויות ומריץ את השרת. פתח בדפדפן: **http://localhost:8000**

> בלי מפתח API האפליקציה עדיין עובדת — פשוט מדלגים על החילוץ האוטומטי ומזינים נתונים ידנית, ואז מורידים אקסל/מצגת.

## איך משתמשים
1. **גרור דוחות** (PDF / תמונות) — אפשר כמה שנים יחד.
2. לחץ **"עבד דוחות עם AI"** — Claude קורא ומאחד את כל השנים לציר אחד וממלא את השדות.
3. **בדוק וערוך** את מה שחולץ (וכל שדה נוסף שתרצה — שוק, צוות, תנאי עסקה וכו').
4. **הורד**: אקסל מלא · מצגת · או חבילת ZIP (אקסל + מצגת + `pitch-data.json`).

האקסל מכיל נוסחאות חיות; Excel/LibreOffice מחשבים אותן אוטומטית בפתיחה.

## הגדרות (משתני סביבה)
- `ANTHROPIC_API_KEY` — חובה לחילוץ אוטומטי.
- `ANTHROPIC_MODEL` — ברירת מחדל `claude-sonnet-4-6`; אפשר לשנות למודל אחר.

## Docker (אופציונלי)
```bash
docker build -t deal-studio .
docker run -p 8000:8000 -e ANTHROPIC_API_KEY=sk-ant-xxxxx deal-studio
```

## העלאה לאונליין — גישה ללקוחות ושותפים בלבד

האפליקציה כוללת **מסך כניסה עם סיסמה**. הכניסה מופעלת אוטומטית ברגע שמגדירים `APP_PASSWORD`
(מקומית, בלי המשתנה הזה, היא נשארת פתוחה ונוחה). בנוסף יש הגבלת קצב להגנה על חשבון ה-API.

**משתני סביבה לפריסה:**
| משתנה | תפקיד |
|-------|-------|
| `ANTHROPIC_API_KEY` | מפתח ה-API (סוד) — חובה לחילוץ |
| `APP_PASSWORD` | סיסמת הגישה שתחלק ללקוחות/שותפים |
| `SESSION_SECRET` | מחרוזת אקראית לחתימת ה-cookie (המארח יכול לייצר) |
| `COOKIE_SECURE` | `1` בפרודקשן (cookie נשלח רק ב-HTTPS) |
| `MAX_EXTRACTS_PER_HOUR` | תקרת חילוצים לשעה לכל IP (ברירת מחדל 60) |
| `ANTHROPIC_MODEL` | מודל (ברירת מחדל `claude-sonnet-4-6`) |

**הדרך הפשוטה — Render (יש קובץ `render.yaml` מוכן):**
1. העלה את התיקייה למאגר ב-GitHub (פרטי).
2. ב-render.com → **New → Blueprint** → חבר את המאגר. Render בונה אוטומטית מה-Dockerfile.
3. בהגדרות השירות הזן את הסודות: `ANTHROPIC_API_KEY` ו-`APP_PASSWORD`. (השאר מוגדר ב-`render.yaml`.)
4. תוך דקות תקבל כתובת `https://deal-studio-xxx.onrender.com` עם HTTPS. שלח אותה + הסיסמה ללקוחות/שותפים.

> עובד באותו אופן ב-Railway / Fly.io / Google Cloud Run — כולם קוראים `Dockerfile` ומאזינים ל-`$PORT`.
> להחלפת סיסמה: שנה את `APP_PASSWORD` (כל מי שהיה מחובר יתבקש להתחבר מחדש).

## הערות
- **פרטיות**: הדוחות נשלחים ל-Anthropic API לצורך החילוץ בלבד; הם לא נשמרים בשרת (עיבוד בזיכרון).
- **שפה**: הממשק בעברית; תוכן התוצרים באנגלית (סטנדרט M&A).
- **דיוק**: ה-AI מחלץ מה שמופיע בדוח (פיננסים, סגמנטים, מרווחים). שדות שאינם בדוח (שוק, צוות, תנאי עסקה) ממולאים בערכי ברירת מחדל הניתנים לעריכה במסך הסקירה.
